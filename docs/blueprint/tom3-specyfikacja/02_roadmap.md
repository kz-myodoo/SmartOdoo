## Jak czytać tę roadmapę?

Roadmapa to **strategiczny plan** projektów i funkcji, które realizujemy. Każdy cel ma:
- **Status:** 🔴 Backlog → 🟡 In Progress → 🟢 Done
- **Sprinty:** Lista sprintów, które realizują ten cel (link do `docs/sprints/`)
- **Priorytet:** P0 (krytyczny) → P1 (ważny) → P2 (nice-to-have)

Sprinty ZAWSZE odnoszą się do jednego z celów poniżej. Jeśli tworzysz nowy sprint — POWIĄŻ go tutaj.
